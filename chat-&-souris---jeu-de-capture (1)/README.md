# 🐾 Chat & Souris — Jeu de Capture

**Chat & Souris** est un jeu d'arcade amusant où vous incarnez un chat essayant de capturer toutes les souris.

## 🎮 Le Concept
Le but est simple : capturez toutes les souris du niveau avant que votre jauge d'énergie (faim) ne tombe à zéro.

### Points Forts :
- **Gameplay Simple** : Facile à prendre en main, difficile à maîtriser.
- **Niveaux Progressifs** : La difficulté augmente à mesure que vous progressez.
- **Score & Leaderboard** : Battez vos propres records et ceux de vos amis.

## 🛠️ Stack Technique
- **React 19** & **TypeScript**
- **HTML5 Canvas**
- **Tailwind CSS**

## 🚀 Installation
1. Clonez le dépôt.
2. Lancez `npm install` puis `npm run dev`.

## 🕹️ Commandes
- **Souris / Tactile** : Dirigez le chat en déplaçant le curseur.
- **Clic / Tap** : Effectuez un Dash pour accélérer.
